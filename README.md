# 🤖 Machine Learning Tasks Collection

## 📌 Overview

This repository contains a collection of **20 Machine Learning tasks** covering the complete pipeline from **data preprocessing to model building and evaluation**.

The project is designed to provide hands-on experience in:

* Data cleaning and preparation
* Exploratory Data Analysis (EDA)
* Supervised Learning
* Unsupervised Learning

---

## 🎯 Objectives

* Apply data preprocessing techniques on real datasets
* Perform data analysis to extract meaningful insights
* Build and evaluate machine learning models
* Compare different algorithms and approaches
* Understand the full ML workflow

---

## 🧠 Tasks Breakdown

### 🔹 Data Preprocessing

1. Handle missing values
2. Remove duplicates
3. Data cleaning
4. Feature scaling (Normalization & Standardization)
5. Encode categorical variables

---

### 🔹 Exploratory Data Analysis (EDA)

6. Summary statistics
7. Data visualization (histograms, boxplots)
8. Correlation analysis (heatmaps)
9. Outlier detection
10. Feature distribution analysis

---

### 🔹 Supervised Learning

11. Linear Regression
12. Logistic Regression
13. Decision Tree
14. Random Forest
15. Model evaluation (Accuracy, Precision, Recall, F1-score)

---

### 🔹 Unsupervised Learning

16. K-Means Clustering
17. Hierarchical Clustering
18. Dimensionality Reduction (PCA)

---

### 🔹 Additional Tasks

19. Feature selection and importance analysis
20. Model comparison and performance optimization
